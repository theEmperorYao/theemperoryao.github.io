public static int subtractProductAndSum2(int n) {
    int prod = 1;
    int sum = 0;
    while (n >0){
        prod *= (n % 10);
        sum += (n % 10);
        n=n/10;
    }
    return prod-sum;
}

4.ListNode反转链表（递归法）

（图是leetcode上面其他同学制作的，做的真棒）

public static ListNode reverseList(ListNode head) {
    if (head == null || head.next == null) return head;
    //p指向最后一个结点
    ListNode p = reverseList(head.next);
    head.next.next = head;
    head.next = null;
    return p;
}

思想：指针p通过递归到找最后一个ListNode结点，head.next==null,目的是查找倒数第二个结点，这样才可以让head（此时指向倒数第二个结点）.next.next=head,也就是完成让 最后一个结点指向倒数第二个结点的过程，然后置空，注意，至此以后指针p一直会指向最后一个结点,然后依次回溯对每一个结点进行指针的反转。

5.将A数组与B数组合并

A数组的容量可以装下所有元素

假设A[1,2,3,4,5,6] B[2,5,6]

采用指针倒序遍历的方式

public static void merge2(int[] A, int m, int[] B, int n) {
    for (int cur= m-- + n-- -1;cur>=0;cur--){
        if ( m<0){
            A[cur]=B[n--];
        }else if (n<0){
            A[cur]=A[m--];
        }else {
            A[cur]=A[m]>B[n]?A[m--]:B[n--];
        }
    }
}
